## Memory Pruning & Abstraction

- **Event-based Memory**: Only record metadata when a **state transition** occurs (e.g., "Door opened" vs. continuous video feed).
- **Semantic Abstraction**: Compressing complex physics interactions into atomic semantic actions (e.g., "Grasped the mug").
- **MCP Integration**: Protocols for efficiently streaming robot SDK states (joint angles, torque) to the agent's memory hub.

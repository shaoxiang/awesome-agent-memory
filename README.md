## Foundational Theories & Surveys
*A deep dive into the cognitive science and architectural theory behind agent memory.*

- [**A Survey on Memory for LLM-based Agents**](https://arxiv.org/abs/2411.13555) - A comprehensive survey on mechanisms, evaluations, and architectures.
- [**A Survey on the Security of Long-Term Memory in LLM Agents: Toward Mnemonic Sovereignty**](http://arxiv.org/abs/2604.16548v1) - This survey establishes a comprehensive security framework specifically for Agent_Memory, addressing vulnerabilities across its entire lifecycle from initial writing and persistent storage to retrieval and systematic rollback. By cross-tabulating six distinct memory phases with four core security objectives, the authors systematically categorize emerging threats like cross-session poisoning, unauthorized extraction, and control-flow hijacking in persistent LLM systems. The analysis identifies critical research gaps in confidentiality and governance primitives, providing engineers with a structured blueprint for designing secure, resilient, and governable long-term memory architectures for autonomous agents.
- [**Hybrid Event Memory as a Case Base for State Estimation in Cognitive Agents**](https://openalex.org/W4385384942) - This paper introduces a hybrid event memory system that functions as a case base for state estimation in cognitive agents. By structuring past events as retrievable cases, the approach directly addresses agent memory for context-aware reasoning and decision-making. It proposes a memory architecture that integrates episodic and semantic elements, enabling agents to leverage historical experiences for improved state estimation. This work contributes to the development of memory modules for intelligent agents, aligning with the Agent_Memory focus on explicit memory structures in cognitive architectures.
- [**Evolving-LLM-Agent-Memory-Survey**](https://github.com/FeishuLuo/Evolving-LLM-Agent-Memory-Survey) - This is a curated list of papers from the survey 'From Storage to Experience: A Survey on the Evolution of LLM Agent Memory Mechanisms'. It explicitly targets agent memory, covering reflection, self-improvement, and other memory paradigms. The repo serves as a comprehensive resource for researchers exploring the progression of memory systems in LLM-based agents. It is a reference collection, not a tool, but directly addresses the core topic.
- [**To Know is to Construct: Schema-Constrained Generation for Agent Memory**](http://arxiv.org/abs/2604.20117v1) - This paper introduces SCG-MEM, a novel schema-constrained generative architecture that fundamentally restructures Agent_Memory access by replacing traditional dense retrieval with cognitively inspired generative recall. By maintaining a dynamic cognitive schema and an associative graph, the system strictly constrains LLM decoding to prevent structural hallucinations while supporting continuous adaptation through assimilation and accommodation mechanisms. Comprehensive evaluations demonstrate that this approach significantly improves retrieval precision and contextual coherence in multi-step reasoning tasks, offering a robust, theoretically grounded framework for deploying reliable long-term memory systems in autonomous AI agents.
- [**The Continuity Layer: Why Intelligence Needs an Architecture for What It Carries Forward**](http://arxiv.org/abs/2604.17273v1) - Introducing the continuity layer, this position paper proposes a novel architectural primitive designed to resolve cross-session amnesia in AI systems by advancing Agent_Memory beyond flat retrieval mechanisms. The authors develop Decomposed Trace Convergence Memory, a storage method utilizing write-time decomposition and read-time reconstruction to maintain persistent contextual understanding across extended interactions. Validated against the ATANT benchmark and existing agentic-memory frameworks, this architecture provides a structured pathway for developers to implement robust, long-horizon state retention in autonomous agents, ultimately enabling more coherent and self-improving AI workflows.
- [**Memory in the LLM Era: Modular Architectures and Strategies in a Unified Framework**](http://arxiv.org/abs/2604.01707v1) - This paper introduces a unified modular framework that systematically categorizes and evaluates existing agent memory architectures for LLM-based systems. By benchmarking representative strategies across two established datasets, the authors identify critical performance bottlenecks and design a novel hybrid memory method that surpasses current state-of-the-art approaches. The study explicitly centers on Agent_Memory optimization, detailing how modular composition enables knowledge accumulation and iterative self-evolution in long-horizon tasks. Practically, this work provides developers with a standardized evaluation protocol and a highly adaptable memory blueprint for building more robust, self-improving AI agents.
- [**Mind Your HEARTBEAT! Claw Background Execution Inherently Enables Silent Memory Pollution**](http://arxiv.org/abs/2603.23064v3) - This paper identifies a critical security vulnerability in personal AI agents where heartbeat-driven background execution silently pollutes both short-term context and long-term agent memory. By formalizing an Exposure-Memory-Behavior pathway and evaluating it through the MissClaw experimental platform, the authors demonstrate how untrusted external content bypasses user awareness to durably alter downstream agent responses. The study explicitly analyzes agent memory persistence mechanisms and cross-session contamination rates, providing essential architectural guidelines for securing memory isolation, implementing dynamic memory allocation safeguards, and ensuring provenance tracking in autonomous AI systems.
- [**LLM Agent Memory: A Survey from a Unified Representation–Management Perspective**](https://openalex.org/W7140163562) - This survey establishes a comprehensive taxonomy for LLM Agent Memory by decoupling representation paradigms from systematic management workflows. It explicitly categorizes Agent_Memory architectures into natural language tokens, intermediate representations, and parameter-based storage, while standardizing construction, update, and retrieval stages to clarify critical design trade-offs and scalability constraints across diverse implementations. Software architects and AI researchers gain a structured blueprint for selecting, optimizing, and benchmarking long-context memory modules, directly accelerating the engineering of robust, multi-turn autonomous systems capable of sustained reasoning.
- [**TERRAIN MEMORY: LEVERAGING THE FULL BREADTH OF LLM TRAINING FOR EXPERIENTIAL AI AGENT MEMORY**](https://openalex.org/W7136613094) - This paper introduces Terrain Memory, a novel experiential agent memory architecture that encodes task experiences as multi-dimensional constructs anchored to abstract landscape archetypes. Unlike traditional factual retrieval systems, it explicitly models agent memory through dynamic dimensions like terrain, weather, sensations, and risks, implementing a living lifecycle where memories strengthen, fade, or accumulate based on continuous interaction. By leveraging the broader semantic and emotional breadth of LLM training data, this approach transforms static recall into adaptive experiential learning, offering a robust framework for building more context-aware and self-improving AI agents.
- [**SuperLocalMemory V3: Information-Geometric Foundations for Zero-LLM Enterprise Agent Memory**](http://arxiv.org/abs/2603.14588v1) - This paper introduces a rigorous information-geometric framework specifically designed to overhaul enterprise Agent_Memory architectures by replacing heuristic retrieval with formal mathematical guarantees. It substitutes cosine similarity with Fisher information metrics, models memory lifecycle management via Riemannian Langevin dynamics, and employs sheaf cohomology to automatically detect irreconcilable contradictions across stored contexts. Evaluated on the LoCoMo benchmark, the proposed system achieves significant accuracy improvements over engineering baselines while eliminating manual decay tuning, thereby delivering provable consistency and scalable retrieval for complex enterprise Agent_Memory deployments.
- [**Joint Optimization of Multi-agent Memory System**](http://arxiv.org/abs/2603.12631v2) - The paper proposes CoMAM, a joint optimization framework for multi-agent memory systems in LLMs, addressing limitations of independent agent optimization. By modeling the multi-agent pipeline as an MDP and using end-to-end reinforcement learning with an adaptive credit assignment mechanism, agents co-adapt through a combination of local task rewards and an adaptively weighted global reward. This approach enhances inter-agent collaboration, leading to consistent outperformance over leading memory systems. The work directly advances Agent Memory research by providing a principled method for optimizing complex memory infrastructures, which is essential for long-horizon, multi-turn interactions.
- [**Governing Evolving Memory in LLM Agents: Risks, Mechanisms, and the Stability and Safety Governed Memory (SSGM) Framework**](http://arxiv.org/abs/2603.11768v1) - This paper introduces the Stability and Safety-Governed Memory (SSGM) framework, explicitly designed to govern evolving Agent_Memory within autonomous LLM-based systems. It tackles critical operational risks like semantic drift and topology-induced knowledge leakage by decoupling memory consolidation from agent execution via consistency verification, temporal decay modeling, and dynamic access control protocols. By formalizing a comprehensive taxonomy of memory corruption vulnerabilities, the research delivers a practical governance architecture that ensures secure, persistent, and reliable long-term memory deployment across complex, real-world agentic workflows and continuous learning pipelines.
- [**Multi-Agent Memory from a Computer Architecture Perspective: Visions and Challenges Ahead**](http://arxiv.org/abs/2603.10062v2) - This position paper introduces a novel computer architecture-inspired framework for collaborative AI systems, explicitly addressing Agent_Memory scalability and consistency challenges. The authors propose a comprehensive, modular three-layer hierarchical structure comprising I/O, cache, and persistent memory tiers, while identifying critical protocol gaps in cross-agent cache sharing and structured access control to formalize shared versus distributed paradigms. By establishing these foundational, actionable engineering principles, the work directly guides developers in designing robust, high-throughput Agent_Memory infrastructures for complex, reliable, scalable, and long-running multi-agent deployments.
- [**Memory for Autonomous LLM Agents:Mechanisms, Evaluation, and Emerging Frontiers**](http://arxiv.org/abs/2603.07670v1) - This comprehensive survey formalizes agent memory as a tightly coupled write–manage–read loop and introduces a detailed three-dimensional taxonomy spanning temporal scope, representational substrates, and dynamic control policies. It systematically examines five core Agent_Memory mechanism families, including retrieval-augmented stores, reflective self-improvement, and policy-learned management, while rigorously evaluating multi-session benchmarks that interleave recall with complex decision-making. By addressing critical engineering realities like contradiction handling and strict latency budgets, the paper delivers a practical architectural blueprint for developers constructing robust, long-horizon autonomous systems.
- [**Memory as Ontology: A Constitutional Memory Architecture for Persistent Digital Citizens**](http://arxiv.org/abs/2603.04740v1) - This paper introduces the Memory-as-Ontology paradigm and proposes Animesis, a novel Agent_Memory system explicitly built on a Constitutional Memory Architecture. It fundamentally shifts development focus from functional data retrieval to existential continuity by implementing a comprehensive four-layer governance hierarchy alongside a dynamic multi-layer semantic storage mechanism. By prioritizing long-term identity persistence and cognitive self-regulation across underlying model upgrades, the framework establishes a robust architectural standard for engineering self-sustaining digital citizens and continuously evolving autonomous AI companions in complex, real-world deployment scenarios.
- [**Semantic XPath: Structured Agentic Memory Access for Conversational AI**](http://arxiv.org/abs/2603.01160v1) - This paper introduces Semantic XPath, a novel tree-structured Agent_Memory module explicitly designed to efficiently access and update dynamic, long-term conversational states. By replacing flat retrieval mechanisms with hierarchical query paths, comprehensive experiments demonstrate a 176.7% performance gain over standard flat-RAG baselines while consuming merely 9.1% of the tokens required by traditional in-context approaches. The accompanying interactive demo successfully validates this architecture, offering practitioners a highly scalable framework for engineering next-generation task-oriented agents with robust, structured memory management capabilities for real-world deployment.
- [**Graph-based Agent Memory: Taxonomy, Techniques, and Applications**](http://arxiv.org/abs/2602.05665v1) - This comprehensive survey systematically reviews graph-based Agent_Memory as a foundational architecture for enabling long-horizon reasoning and continuous self-evolution in modern LLM-driven systems. It establishes a detailed taxonomy and analyzes full-lifecycle techniques for memory extraction, structured storage, efficient retrieval, and dynamic evolution, while cataloging specialized open-source libraries and rigorous evaluation benchmarks. By synthesizing graph-centric methodologies and outlining critical implementation challenges, this work provides developers with actionable architectural blueprints for building robust, knowledge-accumulating, highly scalable autonomous agents capable of complex task execution.
- [**Cognitive Modeling for Long-Horizon Agent Learning via Integrated Long-Term Memory and Reasoning**](https://openalex.org/W7147040153) - The paper proposes a cognitive modeling framework that integrates long-term memory and reasoning to improve agent performance in long-horizon sequential tasks. It introduces a structured long-term memory mechanism for continuous storage and selective updating of key information, coupled with a memory retrieval-driven reasoning module that directly informs decision logic. By tightly coupling perception, memory management, reasoning, and policy generation into an end-to-end cognitive loop, the approach mitigates long-range dependency and partial observability issues. Experiments demonstrate superior task completion quality and decision efficiency, highlighting the value of unified memory and reasoning for long-horizon agent learning.
- [**E-mem: Multi-agent based Episodic Context Reconstruction for LLM Agent Memory**](http://arxiv.org/abs/2601.21714v1) - This paper introduces E-mem, a novel framework that redefines Agent_Memory by shifting from destructive preprocessing to episodic context reconstruction. Leveraging a heterogeneous hierarchical architecture, the system employs multiple assistant agents to preserve uncompressed memory contexts while a master agent coordinates global planning, with evaluations on the LoCoMo benchmark demonstrating significant performance gains over existing methods. By enabling local reasoning within activated memory segments, this approach delivers substantial improvements in logical accuracy and computational efficiency, offering a scalable solution for advanced deliberative AI systems.
- [**BMAM: Brain-inspired Multi-Agent Memory Framework**](http://arxiv.org/abs/2601.20465v1) - This paper introduces BMAM, a brain-inspired framework that explicitly models Agent_Memory as functionally specialized subsystems operating across complementary time scales to prevent long-horizon consistency degradation. By decomposing storage into episodic, semantic, salience-aware, and control-oriented components, the architecture organizes past experiences along explicit timelines and retrieves contextual evidence through multi-signal fusion. Evaluated on the LoCoMo benchmark with 78.45% accuracy, the approach demonstrates how hippocampus-inspired episodic retrieval significantly enhances temporal reasoning, offering a scalable foundation for building temporally stable AI agents.
- [**Rethinking Memory Mechanisms of Foundation Agents in the Second Half: A Survey**](http://arxiv.org/abs/2602.06052v3) - This comprehensive survey establishes a unified taxonomy for foundation agent memory, categorizing mechanisms across internal/external substrates, cognitive types like episodic and semantic memory, and agent-centric versus user-centric subjects. It explicitly focuses on Agent_Memory as a core architectural component, analyzing how memory operations integrate with diverse agent topologies and learning policies for long-horizon tasks. The authors systematically review evaluation benchmarks and metrics designed to measure memory utility in dynamic environments. By synthesizing hundreds of recent studies, this work provides practitioners with a structured roadmap for designing robust, scalable memory systems that bridge the gap between theoretical models and real-world agent deployment.
- [**The AI Hippocampus: How Far are We From Human Memory?**](http://arxiv.org/abs/2601.09113v1) - This comprehensive survey presents a structured taxonomy of memory mechanisms in modern large language models, categorizing existing approaches into implicit, explicit, and agentic paradigms. It systematically reviews how Agent_Memory enables persistent, temporally extended structures for autonomous systems, supporting long-term planning, self-consistency, and multi-agent collaboration through dynamic storage frameworks. While offering valuable theoretical insights and a rigorous literature synthesis, it remains a broad academic overview rather than introducing a dedicated Agent_Memory method, dataset, or experimental platform for direct developer implementation.
- [**From Storage to Experience: A Survey on the Evolution of LLM Agent Memory Mechanisms**](https://openalex.org/W7119539565) - This comprehensive survey introduces a unified evolutionary framework specifically for Agent_Memory, categorizing architectural development into Storage, Reflection, and Experience stages. It explicitly examines how memory systems transition from basic trajectory preservation to advanced cross trajectory abstraction, directly tackling long range consistency and continual learning demands within dynamic environments. By bridging cognitive science theories with practical system engineering, the paper delivers concrete design principles and a clear implementation roadmap for developers constructing next generation autonomous agents with robust, self evolving memory capabilities and highly optimized context management.
- [**AI Meets Brain: Memory Systems from Cognitive Neuroscience to Autonomous Agents**](http://arxiv.org/abs/2512.23343v1) - This comprehensive survey bridges cognitive neuroscience and autonomous agents by systematically analyzing biological and artificial memory mechanisms across multiple disciplines. It explicitly examines Agent_Memory through a comparative taxonomy, detailing storage architectures, management lifecycles, and evaluation benchmarks for LLM-driven systems while addressing security vulnerabilities and multimodal future directions. Although it provides a valuable theoretical framework for designing cognitive architectures, the work remains a literature review without introducing a concrete Agent_Memory implementation, dataset, or experimental platform for direct practical deployment in real-world applications.
- [**Event Extraction in Large Language Model**](http://arxiv.org/abs/2512.19537v1) - This comprehensive survey systematically reviews event extraction methodologies for large language models, tracing their evolution from rule-based systems to generative frameworks while highlighting extraction taxonomies, decoding strategies, and multimodal datasets. Although the authors briefly suggest that structured event stores could theoretically support updatable episodic and agent memory beyond standard context windows, the work fundamentally lacks dedicated memory architectures, concrete implementations, or experimental validations. Consequently, it offers valuable insights for information structuring but provides no direct mechanisms or benchmarks for advancing practical agent memory systems.
- [**Improving Language Agents through BREW**](http://arxiv.org/abs/2511.20297v1) - This paper introduces BREW, a novel framework that optimizes language agents by constructing and refining structured experiential memory instead of relying on computationally expensive policy gradients. The authors explicitly design a partitioned agent memory system enabling efficient retrieval, continuous refinement, and robust noise filtering through task graders and state-space search, validated extensively on OSWorld and SpreadsheetBench. By replacing traditional reinforcement learning with dynamic knowledge base management, this work delivers a highly scalable pathway for building self-improving agents with practical long-term memory capabilities.
- [**Exploitation Is All You Need... for Exploration**](http://arxiv.org/abs/2508.01287v1) - This paper demonstrates that meta-reinforcement learning agents trained exclusively on greedy objectives develop emergent exploratory behaviors when recurring environmental structures and Agent Memory align. By treating Agent Memory as a critical architectural prerequisite, the authors utilize recurrent context windows to retain historical interaction data across stochastic bandit and gridworld experiments, proving through controlled ablations that removing memory entirely eliminates information-seeking policies. These findings offer practical guidelines for designing lightweight meta-RL systems that leverage standard memory modules to achieve efficient exploration without engineering complex intrinsic rewards.
- [**A Survey on the Memory Mechanism of Large Language Model-based Agents**](https://openalex.org/W4412203333) - This paper presents a comprehensive survey explicitly dedicated to Agent_Memory mechanisms in large language model-based autonomous systems, systematically categorizing architectural design patterns, evaluation protocols, and implementation strategies for long-term and self-evolving modules. By synthesizing scattered research into a unified taxonomy and maintaining an active open-source repository, the work establishes foundational principles for engineering robust cognitive architectures. The survey delivers immediate practical value by guiding developers in selecting optimal memory frameworks for complex agent-environment interactions while outlining critical research gaps for next-generation adaptive systems.
- [**Episodic Memory in AI Agents Poses Risks that Should be Studied and Mitigated**](https://openalex.org/W4410609113) - This conceptual paper examines the integration of episodic memory into AI agents, highlighting both its strategic planning capabilities and the novel safety risks it introduces. Rather than implementing a technical Agent_Memory system or dataset, the authors analyze conceptual vulnerabilities like privacy leakage and behavioral drift, outlining four principles to guide safe development. While lacking direct implementation or experimental validation, the work provides crucial ethical guardrails for engineers designing robust long-term memory modules in future autonomous systems and robotics applications.
- [**From LLM to Conversational Agent: A Memory Enhanced Architecture with Fine-Tuning of Large Language Models**](https://openalex.org/W4390721568) - RAISE enhances LLM-based conversational agents with a dual-component memory system (short-term and long-term) for maintaining context in multi-turn dialogues. It fine-tunes GPT-4 using conversation selection, scene extraction, and CoT completion, then evaluates in a real estate sales context. The paper focuses on dialogue agent memory augmentation, not embodied AI or physical environment memory.
- [**Large Language Model-Based Planning Agent with Generative Memory Strengthens Performance in Textualized World**](https://openalex.org/W4401356997) - This paper presents a large language model-based planning agent that leverages generative memory to enhance performance in textualized environments. The proposed generative memory mechanism enables the agent to retain, recall, and synthesize past experiences, improving long-horizon planning and decision-making. Experiments in textualized worlds demonstrate significant gains in task completion and efficiency. This work directly contributes to Agent Memory by introducing a novel memory architecture for LLM agents, with practical value for embodied AI and autonomous planning systems.
- [**Empowering Working Memory for Large Language Model Agents**](https://openalex.org/W4390489503) - This paper applies cognitive psychology's working memory frameworks to enhance LLM agents memory. It proposes a centralized Working Memory Hub and Episodic Buffer to retain memories across sequential interactions, overcoming LLMs' limited memory retention. This architecture enables nuanced contextual reasoning for complex tasks and collaborative scenarios. The paper provides a strategic blueprint for developing LLM agents with human-like memory capabilities, directly contributing to Agent_Memory by designing a memory architecture that integrates central working memory and episodic access.

Targeted at developers building AI agents requiring durable, cross-session memory.
